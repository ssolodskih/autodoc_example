Bases: ABC

Model class. Performs all the actions necessary to convert input data to desired result. These include: - data type checks according to the client-specific data format, - conversion to model internal format, - calculations based upon these data, - data visualization (optional), - storage of iinput, intermediate, and output data as well as error information (optional)
